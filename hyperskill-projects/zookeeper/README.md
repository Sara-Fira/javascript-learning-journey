# Zookeeper

This is a beginner-friendly JavaScript project from Hyperskill.

The program helps zoo staff check different animal habitats. It displays ASCII art of animals and short messages about them.

## What the project does

The project was built in 4 stages:

1. Prints simple text messages.
2. Displays an ASCII image of an animal.
3. Lets the user choose a habitat by entering its number.
4. Runs in a loop until the user types `exit`.

## What I practiced

* Printing text to the console
* Working with strings
* Storing data in an array
* Using user input
* Accessing array elements by index
* Creating a loop
* Using a condition to exit the program
