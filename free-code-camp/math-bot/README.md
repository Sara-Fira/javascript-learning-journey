# MathBot Workshop

A beginner JavaScript project from freeCodeCamp.

This project practices using the JavaScript `Math` object by building a simple MathBot that prints messages and math results to the console.

## What I practiced

- `Math.random()`
- `Math.floor()`
- `Math.ceil()`
- `Math.round()`
- `Math.max()`
- `Math.min()`
- Generating random numbers
- Logging results with `console.log()`

## Project status

Completed as part of my JavaScript learning journey.
