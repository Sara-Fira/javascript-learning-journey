# 📖 Sentence Maker

A simple JavaScript project created as part of the freeCodeCamp curriculum.

In this exercise, I built a small story generator using variables and template-based sentences. 
The program creates two different stories by reassigning values and outputting formatted text to the console.

## 🚀 What this project does

- Stores parts of a story in variables
- Builds sentences using a fixed template
- Reuses variables with new values to generate a second story
- Prints results to the console

## 🧠 What I practiced

- JavaScript variables (`let`)
- String interpolation / template logic
- Variable reassignment
- Console output
- Basic program structure
