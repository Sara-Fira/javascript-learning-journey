# Celsius to Fahrenheit Converter

This is a beginner JavaScript lab from freeCodeCamp.

The goal of this project is to create a function that converts temperature from Celsius to Fahrenheit.

## Formula

```js
fahrenheit = celsius * (9 / 5) + 32;
```

## What it does

The function takes a temperature in Celsius and returns the temperature in Fahrenheit.

Example:

```js
convertCtoF(20);
```

Output:

```js
68
```
