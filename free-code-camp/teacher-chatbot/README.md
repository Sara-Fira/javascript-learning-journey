# Teacher Chatbot

A beginner JavaScript project built as part of the freeCodeCamp curriculum.

## About the Project

Teacher Chatbot is a simple console-based JavaScript project that introduces basic string operations through chatbot-style messages.
The bot presents short learning messages and demonstrates how to work with strings in JavaScript.

## What I Practiced

* Declaring variables
* Using `console.log()`
* Creating strings
* Using template literals
* Reading the `.length` property
* Accessing characters with bracket notation
* Getting the last character of a string
* Using the `indexOf()` method
* Understanding that `indexOf()` is case-sensitive

## Technologies Used

* JavaScript

## What the Program Does

The chatbot introduces itself, explains that the lesson is about JavaScript strings, and shows examples of:

* checking the length of a string,
* accessing the first, second, and last characters,
* finding the position of substrings inside a sentence.
