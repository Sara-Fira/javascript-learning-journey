# Email Masker

This is a beginner JavaScript lab from freeCodeCamp.

The goal of this project is to create a function that masks the username part of an email address with asterisks.

## What it does

The function keeps the first and last character of the email username visible and replaces the middle characters with `*`.

Example:

```js
maskEmail("apple.pie@example.com");
```

Output:

```js
"a*******e@example.com"
```
