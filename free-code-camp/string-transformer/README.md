# String Transformer App

A beginner-friendly JavaScript project from freeCodeCamp.

## About

This project practices basic JavaScript string methods by transforming simple sentences in the console.

## What I Practiced

* Using `replace()` to replace the first matching word
* Using `replaceAll()` to replace all matching words
* Using `repeat()` to repeat text
* Using `trimEnd()` to remove extra space
* Logging results with `console.log()`

## Methods Used

```js
replace()
replaceAll()
repeat()
trimEnd()
```

## Final Result

The project creates a sentence like:

```js
I love love love learning.
```
