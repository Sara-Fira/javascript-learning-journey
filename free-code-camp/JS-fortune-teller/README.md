# Fortune Teller

A beginner JavaScript project from freeCodeCamp.

This project randomly selects one fortune from five possible messages and displays it in the console.

## What I practiced

- Creating string variables
- Using `Math.random()`
- Using `Math.floor()`
- Generating random numbers
- Using `if` statements
- Logging output with `console.log()`

Completed as part of my JavaScript learning journey.
