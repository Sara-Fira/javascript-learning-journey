# Calculator App

A beginner JavaScript project from freeCodeCamp.

This project practices working with functions by building a simple calculator that performs basic math operations.

## What I practiced

- Declaring functions
- Using parameters and arguments
- Returning values from functions
- Addition, subtraction, multiplication and division
- Handling division by zero
- Using `Math.pow()`
- Using `Math.sqrt()`
- Logging results with `console.log()`

## Project status

Completed as part of my JavaScript learning journey.
