# Loan Qualification Checker

This is a beginner JavaScript workshop from freeCodeCamp.

The goal of this project is to practice JavaScript conditionals by building a simple loan qualification checker.

## What it does

The app checks whether a user qualifies for a duplex, condo, or car loan based on:

* annual income
* credit score

Depending on the values, the function returns one of the following messages:

* eligible for a duplex, condo, and car loan
* eligible for a condo and car loan
* eligible for a car loan
* not eligible for any loan

## What I practiced

* Creating variables
* Writing functions with parameters
* Using `if`, `else if`, and `else`
* Working with comparison operators
* Using logical AND `&&`
* Returning values from a function
* Logging results to the console
