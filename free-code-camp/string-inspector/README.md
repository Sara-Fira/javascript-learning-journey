# String Inspector

A beginner-friendly JavaScript mini project from freeCodeCamp focused on practicing string methods.

## Description

This project shows how to use two JavaScript string methods: `includes()` and `slice()`.

## What I Practiced

* Working with strings
* Using `includes()` to check if text exists in a string
* Understanding that `includes()` is case-sensitive
* Using `slice()` to extract parts of a string
* Working with string indexes
* Logging results to the console

## Technologies Used

* JavaScript

## What I Learned

I learned how to check if a string contains specific text and how to extract selected parts of a string using indexes.
