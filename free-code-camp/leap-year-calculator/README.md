# Leap Year Calculator

This is a beginner JavaScript lab from freeCodeCamp.

The goal of this project is to create a function that checks whether a given year is a leap year.

## What it does

The function checks if a year follows the leap year rules:

* the year is divisible by `4`
* but years divisible by `100` are not leap years
* unless they are also divisible by `400`

Example:

```js
isLeapYear(2024);
```

Output:

```js
"2024 is a leap year."
```
