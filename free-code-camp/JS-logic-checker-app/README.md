# Conditional Statements Workshop

A beginner JavaScript workshop from freeCodeCamp focused on conditional statements, comparison operators, and truthy/falsy values.

## What I practiced

* Working with boolean values
* Understanding truthy and falsy values
* Using the `Boolean()` function
* Writing `if` statements
* Writing `if...else` statements
* Using comparison operators like `>=`

## Project example

The workshop uses simple variables related to Timmy, such as:

* whether he has a developer job
* whether he is a gamer
* whether he is old enough to drive

## What I learned

I learned how JavaScript checks conditions and runs different code depending on whether a value is `true` or `false`.
