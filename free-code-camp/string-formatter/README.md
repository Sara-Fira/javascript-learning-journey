# String Formatting Workshop

A beginner-friendly JavaScript mini project from freeCodeCamp focused on practicing basic string formatting methods.

## Description

This project shows how to clean and format text in JavaScript using common string methods.

## What I Practiced

* Creating string variables
* Removing spaces with `trim()`
* Removing spaces from the beginning with `trimStart()`
* Removing spaces from the end with `trimEnd()`
* Converting text to uppercase with `toUpperCase()`
* Converting text to lowercase with `toLowerCase()`
* Using `slice()` to extract parts of a string
* Building a simple camelCase word
* Logging results to the console

## Technologies Used

* JavaScript

## What I Learned

I learned how to format strings, remove extra whitespace, change letter casing, and combine string methods to create a new formatted word.
