# 🤖 JavaScript Trivia Bot

A console-based trivia bot created as part of the freeCodeCamp JavaScript curriculum.

This project focuses on practicing core JavaScript concepts by building a bot that shares programming fun facts and interacts through console messages.

## 🚀 Features

- Greeting and farewell messages
- Dynamic text using variables
- Fun coding facts
- Variable reassignment with `let`
- String interpolation and console output

## 🛠️ Built With

- JavaScript (ES6)

## 📌 Purpose

The goal of this project was to strengthen fundamental JavaScript skills through hands-on practice with variables, strings, and dynamic console interactions.
