# Kelvin Weather

A beginner JavaScript project from Codecademy.

## Project Description

This project converts a temperature from Kelvin to Celsius and Fahrenheit.

The program starts with a temperature in Kelvin, converts it to Celsius, then converts it to Fahrenheit using a simple formula. The final Fahrenheit value is rounded down and printed to the console.

## What I Practiced

* JavaScript variables
* `const` and `let`
* Basic math operations
* Temperature conversion formulas
* `Math.floor()`
* Template literals
* `console.log()`

## Formula Used

```js
Celsius = Kelvin - 273
Fahrenheit = Celsius * (9 / 5) + 32
```

## Example Output

```text
The temperature is 68 degrees Fahrenheit.
```
