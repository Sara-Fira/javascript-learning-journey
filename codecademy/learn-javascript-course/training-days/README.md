# Training Days

A beginner JavaScript project from Codecademy.

## Description

This project focuses on fixing variable scope issues in a JavaScript program.

The program randomly selects a race event and calculates how many training days are needed before the event. It also prints a message with the athlete’s name, event, and training time.

## What I practiced

* JavaScript functions
* Variable scope
* Global scope
* Local scope
* Block scope
* Function parameters
* Avoiding duplicate code
* Debugging `ReferenceError`
* Improving code maintainability

## How it works

The program uses:

* `getRandEvent()` to randomly select an event
* `getTrainingDays()` to return the number of training days
* `logEvent()` to print the athlete’s event
* `logTime()` to print the athlete’s training time

## Main lesson

Variables should be declared in the correct scope.

A variable should not be too global if it is only needed inside one function.
A variable should not be too local if it needs to be used in more than one place.
