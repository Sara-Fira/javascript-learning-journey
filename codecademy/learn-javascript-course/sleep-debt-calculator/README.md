# Sleep Debt Calculator

A beginner JavaScript project from Codecademy.

## Description

This project calculates whether the user got enough sleep during the week.

The program compares the actual number of hours slept with the ideal number of sleep hours and shows whether the user:

* got the perfect amount of sleep,
* slept more than needed,
* should get more rest.

## What I practiced

* JavaScript functions
* Function parameters
* `if / else` statements
* Returning values from functions
* Basic calculations
* Calling functions inside other functions

## How it works

The program uses:

* `getSleepHours(day)` to return sleep hours for each day
* `getActualSleepHours()` to calculate total sleep hours for the week
* `getIdealSleepHours()` to calculate ideal weekly sleep hours
* `calculateSleepDebt()` to compare actual sleep with ideal sleep
