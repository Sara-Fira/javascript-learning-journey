# Dog Years

A beginner JavaScript project from Codecademy.

## Project Description

This project converts a human age into dog years.

The program uses a simple formula where the first two years of a dog’s life count as 10.5 dog years each, and every year after that counts as 4 dog years.

## What I Practiced

* JavaScript variables
* `const` and `let`
* Math operators
* Multiplication assignment operator `*=`
* String methods
* `.toLowerCase()`
* Template literals
* `console.log()`

## How It Works

The program:

1. Stores a human age in a variable.
2. Calculates the first two years as dog years.
3. Calculates the remaining years as dog years.
4. Adds both values together.
5. Prints the final result to the console.

## Example Output

```text
My name is sara. I am 30 years old in human years, which is 133 years old in dog years.
```
