# Magic Eight Ball

A beginner JavaScript project from Codecademy.

## Project Description

This project creates a simple Magic Eight Ball program.

The user asks a question, and the program generates a random answer using JavaScript control flow.

## What I Practiced

* JavaScript variables
* `if`, `else if`, and `else`
* Control flow
* Random number generation
* `Math.random()`
* `Math.floor()`
* Template literals
* `console.log()`

## How It Works

The program:

1. Stores the user's name.
2. Stores the user's question.
3. Generates a random number from 0 to 7.
4. Uses control flow to choose one of eight possible answers.
5. Prints the Magic Eight Ball answer to the console.

## Example Output

```text
Will I be rich?
Signs point to yes
```
