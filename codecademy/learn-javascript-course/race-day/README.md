# Race Day

A beginner JavaScript project from Codecademy.

## Project Description

This project creates a simple race registration program.

The program assigns a random race number to a runner and gives them a race time based on their age and registration status.

## What I Practiced

* JavaScript variables
* Boolean values
* Random number generation
* `Math.random()`
* `Math.floor()`
* `if`, `else if`, and `else`
* Comparison operators
* Logical operators
* Template literals

## How It Works

The program checks:

* if the runner is an adult
* if the runner registered early
* if the runner is under 18
* if the runner is exactly 18

Based on these conditions, the program prints the runner’s race number and race time.

## Example Output

```text
1432 - you will race at 9:30 am.
```
