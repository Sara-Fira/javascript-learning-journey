# Rock Paper Scissors

A beginner JavaScript project from Codecademy.

## Project Description

This project creates a simple Rock, Paper, Scissors game.

The user chooses rock, paper, or scissors, and the computer randomly selects its own choice. The program then compares both choices and determines the winner.

## What I Practiced

* JavaScript functions
* Arrow functions
* `return` statements
* `if`, `else if`, and `else`
* Comparison operators
* Logical operators
* `Math.random()`
* `Math.floor()`
* Template literals
* Basic game logic

## How It Works

The program:

1. Gets the user’s choice.
2. Generates a random computer choice.
3. Compares both choices.
4. Prints the result to the console.

## Example Output

```text
User chose: rock
Computer chose: scissors
User won
```
