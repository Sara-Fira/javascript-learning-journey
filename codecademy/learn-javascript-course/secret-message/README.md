# Secret Message

This is a small JavaScript practice project from the Codecademy **Learn JavaScript** course.

## Project Description

The goal of this project was to use JavaScript array methods to transform an array of words into a secret message.

The exercise focused on modifying an existing array by removing, adding, replacing, and joining elements.

## What I Practiced

* Removing the last element with `.pop()`
* Checking array length with `.length`
* Adding elements to the end with `.push()`
* Replacing an element by index
* Removing the first element with `.shift()`
* Adding an element to the beginning with `.unshift()`
* Replacing multiple elements with `.splice()`
* Creating a sentence from an array with `.join()`
