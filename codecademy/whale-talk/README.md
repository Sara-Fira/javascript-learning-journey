# Whale Talk

This is a small JavaScript practice project from the Codecademy **Learn JavaScript** course.

## Project Description

The goal of this project was to translate a phrase into “whale talk” by extracting vowels from a string.

In whale talk, consonants are removed, and the letters `e` and `u` are doubled.

## What I Practiced

* Creating and using arrays
* Working with strings
* Using nested `for` loops
* Comparing characters
* Adding elements to an array with `.push()`
* Joining array elements with `.join()`
* Converting text to uppercase with `.toUpperCase()`

## Technologies Used

* JavaScript

## Example

Input:

```text
turpentine and turtles
```

Output:

```text
UUEEIEEAUUEE
```
