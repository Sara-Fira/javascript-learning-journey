🐱 Cat of the Day Generator
A simple JavaScript project that generates a random cat with different traits.

📌 About the project
This project generates a random cat object each time the function is executed.
The cat has randomly selected characteristics such as name, color, and mood.

The goal of this project was to practice:

working with arrays
functions in JavaScript
objects
Math.random()
basic program structure

🚀 How it works
When you run the program, it:

Picks a random name from an array
Picks a random color from an array
Picks a random mood from an array
Creates a cat object
Prints the result in the console
🧪 Example output

🐱 Cat of the Day:

Name: Max
Color: ginger
Mood: sleepy
