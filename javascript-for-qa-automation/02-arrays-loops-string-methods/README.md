# Session 2: Arrays, Loops and String Methods

This session focused on practising JavaScript basics in a QA testing context.

## Topics covered

* arrays with test users
* array index access
* `for` loops
* reverse loops
* using a function inside a loop
* basic string methods:

  * `.includes()`
  * `.startsWith()`
  * `.endsWith()`
  * `.trim()`
  * `.toLowerCase()`
  * `.toUpperCase()`

## QA context

The exercises used simple testing-related examples such as:

* checking login results
* validating usernames and emails
* checking error messages
* checking URL paths
* validating file extensions
* normalizing text before comparison

## Key takeaway

JavaScript methods such as `.trim()`, `.toLowerCase()` and `.includes()` are useful for writing more reliable test checks, especially when comparing text, URLs, input values and error messages.
