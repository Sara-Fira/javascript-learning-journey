# JavaScript Core Maintenance — Session 4

This session focused on JavaScript objects through QA-style and API-style exercises.

## Topics Practiced

- creating objects
- accessing properties with dot notation
- modifying existing properties
- adding new properties
- deleting properties with `delete`
- checking whether a property exists with the `in` operator
- passing objects as function arguments
- returning a single property from a function
- returning an entire object
- modifying an object inside a function
- using object properties in conditions
- validating multiple object properties
- creating summaries with template literals

## QA-Style Exercises

The exercises included:

- validating API response objects
- checking status codes and response messages
- validating response time
- checking user permissions
- adding validation results to objects
- working with test case objects
- removing temporary debug data
- checking whether error information exists
- creating readable test and response summaries
