# JavaScript Core Maintenance

This folder contains short JavaScript review sessions focused on fundamentals useful in QA and test automation.

The exercises are organized by topic and gradually increase in difficulty.

## Sessions

### 01 — Core Basics and Conditions

Topics:

- variables: `const`, `let`
- data types and `typeof`
- strict comparisons
- truthy and falsy values
- `if/else`
- logical operators
- string methods
- template literals
- combined QA-style conditions

### 02 — Functions, Return and Scope

Topics:

- function declarations
- parameters and arguments
- `return` vs `console.log()`
- returning strings and booleans
- QA-style validation functions
- global, function, and block scope
- variable declaration vs assignment
- variable shadowing
- arrow functions
- implicit return

## Goal

The goal of these sessions is to maintain and strengthen JavaScript fundamentals before moving further into Playwright and test automation.
